# EVERDUSK

A dark Minecraft horror community website built for immersive lore, projects, and events.

## About

Everdusk is a horror-themed Minecraft community focused on:
- immersive storytelling
- dark lore
- custom projects
- mysterious entities
- cinematic atmosphere
- community events

The website acts as the main hub for:
- lore archives
- project showcases
- community updates
- media galleries
- future expansions

# Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/everdusk.git
```

Open the folder:

```bash
cd everdusk
```

Run locally by opening:

```text
index.html
```

---

# Deployment

This project is hosted using GitHub Pages.

To deploy:

1. Push changes to GitHub
2. Open repository settings
3. Enable GitHub Pages
4. Select the main branch
5. Save
